# Sistema de Almoxarifado

API simples em Java com Spring Boot para controle de ferramentas e empréstimos.

## Funcionalidades

- Cadastro de ferramentas
- Cadastro de funcionários
- Empréstimo de ferramentas

## Tecnologias

- Java
- Spring Boot
- Spring Data JPA

## Endpoints

POST /tools  
POST /employees  
PUT /tools/emprestimos

## Como executar

git clone https://github.com/seu-usuario/seu-repositorio.git  
cd nome-do-projeto  
./mvnw spring-boot:run
